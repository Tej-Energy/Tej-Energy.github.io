CycleIQ — Battery Validation Tool
Building
Automated test suite for mixed-capacity, mixed-generation LFP paralleling. Flags unsafe combinations using data-driven thresholds from cycle studies.
Battery Testing & Characterization
Available

Cell Compatibility Checker

SOH Estimator — Python Notebook

Open-source Jupyter notebook for estimating State of Health from OCV-IR data. Get degradation reports and trend visualization from your own test data.
AI Battery Failure Pattern Library

TwinCell — Battery Digital Twin Engine
Exploring
Physics-informed ML model that ingests raw sensor data and builds a live digital twin of each battery cell, predicting remaining useful life in real time.
Refurb.AI — Second-Life Grading System
