// products

Things we're building.

Early-stage but grounded in real engineering. Each product comes from hands-on work with actual battery systems and data.

Building
BattSight — Battery Analytics Dashboard
Python-native analytics platform for Li-ion fleet health monitoring. OCV-IR trending, SOH decay curves, thermal event detection. Built on 96+ real dashboards from Cospowers deployments.
Building
CycleIQ — Parallel Battery Validation Tool
Automated test suite for mixed-capacity, mixed-generation LFP paralleling. Flags unsafe combinations using data-driven thresholds from 100+ cycle validation studies.
Exploring
TwinCell — Battery Digital Twin Engine
Physics-informed ML model that ingests raw sensor data and builds a live digital twin of each battery cell — predicting remaining useful life and failure modes in real time.
Exploring
Refurb.AI — Second-Life Grading System
AI-assisted diagnostic pipeline for retired EV and telecom batteries. Grades packs for second-use routing — residential storage, industrial backup, or responsible recycling.
Planned
GridSense — V2G and BESS Dispatch Optimizer
Real-time AI dispatcher for battery energy storage systems and EV fleets. Reads live market prices, weather, and load data to optimize charge and discharge decisions automatically.
